2.3 Vectors

Vectors are groups of numbers which, when named â€œvectorsâ€, have both a magnitude and direction. The magnitude of the vector corresponds to the length of the vector. The direction of the vector must be relative to a given coordinate system. Within mathematics, vectors are described using a linearly independent basis. A basis is a set of n vectors within a coordinate system. All other vectors can be build using combinations of these n vectors. The term linearly independent signifies that none of the basis vectors are made up of other basis vectors. The general term used to describe these possible combinations of vectors using a basis is called vector space. It is possible to have a vector space build up of any positive amount of basis vectors. However, since we exist in 3 dimensions, in engineering applications we typically operate in a vector space composed of 3 basis vectors.
